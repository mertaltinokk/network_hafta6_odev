# network_hafta6_odev
Kodu GitHub'da buldum (pioneerhfy) linki: https://github.com/heikipikker/IcmpiFlood/blob/master/IcmpiFlood.py orijinal yapısını bozmamak için sadece Python 3'teki print fonksiyonu farklılıklarını düzelttim. Ayrıca verimliliği artırmak amacıyla multiprocessing yapısını da korudum. Ağ paketlerini düşük seviyede manipüle edebilmek için Scapy kütüphanesini kullandım. Saldırıyı güvenli bir ortamda test etmek amacıyla hedef olarak kendi yerel adresimi (127.0.0.1) seçtim. Terminale python icmp_flood_kod.py 127.0.0.1 komutunu yazarak kodu çalıştırdım ve saldırıyı başlattım. Wireshark üzerinden Loopback trafiğini izleyerek saniyede binlerce ICMP paketinin oluşturulup hedefe gönderildiğini canlı olarak doğruladım. Sonuç olarak bu çalışma sayesinde bir ağ protokolünün (ICMP) kötü niyetli kullanım durumunda sistem kaynaklarını nasıl zorlayabileceğini ve siber güvenlikte kaba kuvvet temelli ağ saldırılarının temel mantığını uygulamalı olarak öğrenmiş oldum.
